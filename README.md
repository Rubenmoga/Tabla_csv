# Tabla_csv

Pequeño script para el manejo sencillo de tablas csv. Imprime por terminal la tabla formateada como:

|Nombre|Edad|Ciudad   |<br>
+------+----+---------+<br>
|Ana   |28  |Madrid   |<br>
|Luis  |21  |Palencia |<br>
|Carlos|34  |Barcelona|<br>
|Marta |29  |Sevilla  |<br>

Se puede añadir una o varias nuevas filas desde terminal.
